# Welcome to NetLink

NetLink is a professional networking platform designed to help users connect with professionals, attend events, discover podcasts, and grow their network.

## Project Setup

### Prerequisites

- Node.js & npm installed

### Running locally

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
